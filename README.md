# Лендинг для онлайн-репетиторства

Современный одностраничный сайт для онлайн-репетиторства (5-9 классы).

## Структура

- **Hero-блок**: Заголовок, подзаголовок и кнопка призыва к действию
- **Календарь**: Интеграция с Google Calendar для записи на бесплатный урок
- **Коммуникация**: Кнопки для связи через WhatsApp и Telegram
- **Цены**: Карточка с ценой и кнопкой оплаты через Kaspi Pay
- **Преимущества**: Три карточки с основными преимуществами

## Настройка

### 1. Google Calendar
В файле `index.html` замените `YOUR_CALENDAR_ID` на ваш реальный ID календаря:
- Откройте Google Calendar
- Перейдите в настройки календаря
- Скопируйте ссылку для встраивания или ID календаря

### 2. WhatsApp
Замените `YOUR_PHONE_NUMBER` на ваш номер в международном формате (например: 77001234567)

### 3. Telegram
Замените `YOUR_TELEGRAM_USERNAME` на ваш username в Telegram (без @)

### 4. Kaspi Pay
Замените `YOUR_KASPI_LINK` на вашу ссылку для оплаты через Kaspi

## Запуск

Просто откройте `index.html` в браузере. Сайт готов к использованию!

## Цветовая схема

- Фон: Белый (#fff)
- Акцентный цвет (доверие): Синий/фиолетовый градиент (#667eea, #764ba2)
- Кнопки призыва к действию: Оранжевый (#ff9800)
- Kaspi: Красный (#f14635)
- WhatsApp: Зеленый (#25D366)
- Telegram: Синий (#0088cc)
